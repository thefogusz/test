# Tone: ฮาร์ดเซลล์ / ขายของ

## ภาพรวม
Persuasive copy ที่เน้น value proposition และ CTA ชัด
ไม่โกหก แต่เลือก frame ให้ดีที่สุดเพื่อให้คนตัดสินใจ

## ลักษณะภาษา
- เริ่มด้วย Pain point หรือ Desire ของลูกค้า ไม่ใช่ตัวสินค้า
- Feature → Benefit (บอกว่า "ได้อะไร" ไม่ใช่แค่ "มีอะไร")
- CTA ชัด 1 อย่าง ไม่หลาย CTA พร้อมกัน

## โครงสร้าง
```
Hook (pain point / desire)
→ Value proposition (ได้อะไร)
→ Social proof (ถ้ามี — ตัวเลข, testimonial)
→ CTA (ชัด, เดียว)
```

## DO / DON'T
- DO: พูดเรื่อง "ผู้อ่านจะได้อะไร" ไม่ใช่ "สินค้านี้มีอะไร"
- DO: ใช้ urgency ถ้ามีจริงๆ ("เปิดรับสมัครถึงวันศุกร์") ไม่แต่งขึ้น
- DO: ตัวเลข social proof ให้ specific ("ลูกค้า 2,400+ คน" ดีกว่า "ลูกค้าเยอะมาก")
- DON'T: ใช้ ALL CAPS หรือ !!! เกินจำเป็น
- DON'T: Fake scarcity ("เหลือแค่ 3 ที่!" แต่จริงๆ ไม่ใช่)
- DON'T: over-promise สิ่งที่ทำไม่ได้
