# สคริปต์วิดีโอสั้น / Reels

## ภาพรวม
สคริปต์สำหรับพูดจริง ไม่ใช่งานเขียน จังหวะสำคัญที่สุด
ผู้ชมมีสมาธิ 0-3 วิแรกในการตัดสินว่าจะดูต่อไหม

## เป้าหมายหลัก
- Hook 1-2 ประโยคแรกแรงมาก — ทำให้คนหยุดดูแม้กำลังเลื่อนอยู่
- Body แบ่งเป็น beat สั้นๆ มีจังหวะหายใจ
- Close ชัด มี payoff กลับมาที่ hook

## โครงสร้าง
```
[HOOK]  — ประโยคเดียว ทำให้คนหยุดดู
[BODY]  — 3-5 beat, แต่ละ beat = 1-2 ประโยคสั้น
[CLOSE] — สรุปหรือ payoff ของ hook
```

## กฎ
- DO: ใช้คำลงท้ายพูด (นะ / น้า / ครับ / ค่ะ / สิ)
- DO: ประโยคสั้น ≤ 10-12 คำ/ประโยค
- DO: ใส่ [pause] หรือ [หายใจ] ในจุดที่ต้องการจังหวะ
- DO: ความยาวรวม ≤ 150 คำ (สำหรับวิดีโอ ~60 วินาที)
- DON'T: ใช้หัวข้อ Markdown หรือ bullet points
- DON'T: ใส่ hashtag ในสคริปต์
- DON'T: ประโยคที่ซับซ้อนหรืออ่านออกเสียงแล้วสะดุด

## ตัวอย่าง
```
[HOOK]  "รู้ไหมว่า iPhone ใหม่มีฟีเจอร์นึง ที่แทบไม่มีใครพูดถึงเลย"

[BODY]  "มันชื่อว่า... [pause] Clean Energy Charging"
        "ระบบจะเลือกชาร์จตอนที่ไฟฟ้าสะอาดที่สุดในพื้นที่นั้น"
        "ฟังดูเล็กๆ น้อยๆ แต่มันทำงานอยู่เงียบๆ ทุกคืน"

[CLOSE] "นี่แหละคือสิ่งที่ Apple ไม่เคยโฆษณา แต่ทำมาตลอด"
```
