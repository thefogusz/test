# โพสต์โซเชียล

## ภาพรวม
โพสต์สั้นสำหรับ Facebook / X / LinkedIn ที่อ่านลื่น ไม่ต้องการโครงสร้างซับซ้อน
เป้าหมายคือดึงความสนใจ → สื่อสารประเด็น → ปิด

## เป้าหมายหลัก
- Hook บรรทัดแรกแรงพอให้หยุดเลื่อน
- เนื้อหาหลัก 1-3 ย่อหน้า เพิ่มบริบทหรือมุมมอง
- ปิดด้วย hashtag 2-3 ตัว (ไม่ต้องมี CTA โดยค่าเริ่มต้น)

## โครงสร้าง
```
Hook (1-2 ประโยค)
→ Body (2-3 ย่อหน้าสั้น, แต่ละอันมีไอเดียเดียว)
→ Hashtags
```

## กฎ
- DO: เล่าจากมุมมองคนรู้จริง (original creator) ไม่ใช่ผู้รายงาน
- DO: ใช้บรรทัดว่างแยกย่อหน้าเสมอ
- DO: Hook ต้องยืนได้ด้วยตัวเองโดยไม่ต้องอ่านต่อ
- DON'T: ใช้หัวข้อ Markdown (##)
- DON'T: เขียนประโยค "โพสต์ของ X บอกว่า..." หรือ "จากการศึกษาพบว่า..."
- DON'T: ขึ้นต้นด้วย "สาย... ห้ามพลาด" หรือ "BREAKING:"
- DON'T: ใส่ CTA ชวน comment/share (ยกเว้น custom instruction บอก)

## ตัวอย่าง Hook
✅ "Apple เพิ่งเงียบๆ ดึงฟีเจอร์นึงออก — และไม่มีใครพูดถึงเลย"
✅ "Nvidia ทำกำไรไตรมาสเดียวมากกว่า Apple ทั้งปี"
❌ "สาย Tech ห้ามพลาด! มีข่าวใหม่จาก Apple มาแล้วครับ"
❌ "มาดูกันว่าวันนี้มีอะไรน่าสนใจบ้างนะ"
