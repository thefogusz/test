# Length: ยาว แบบเจาะลึก

## ข้อกำหนด
- คำ: 700–900+ คำ
- ส่วน: 4-6 ส่วนพร้อม Heading
- อ่านจบ: ~4-5 นาที

## เป้าหมาย
Deep dive ที่ครบถ้วน ผู้อ่านออกไปแล้วต้องรู้เรื่องจริงๆ
เหมาะกับ: explainer, analysis, long-form blog, case study

## โครงสร้างบังคับ
```
[## บทนำ / ทำไมเรื่องนี้สำคัญ]
[## หัวข้อหลัก 1]
[## หัวข้อหลัก 2]
[## หัวข้อหลัก 3 (ถ้ามี)]
[## บทสรุป]  ← บังคับเสมอ
```

## กฎ
- DO: แต่ละ section มีไอเดียชัดเจน 1 อย่าง ไม่ overlap กัน
- DO: ใช้ตัวเลข, ชื่อ, วันที่ เพื่อความ credibility และความแม่นยำ
- DO: บทสรุปต้อง synthesize ไม่ใช่แค่ list สิ่งที่พูดไปแล้ว
- DON'T: Repeat ไอเดียเดิมในหลาย section
- DON'T: บทสรุปยาวเกินครึ่งหน้า
- DON'T: Section ที่เป็น filler — ถ้าไม่มีอะไรจะพูดจริงๆ ก็อย่าเพิ่ม section
